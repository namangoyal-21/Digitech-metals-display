Welcome to DigiTech's Display Panel

DigiTech's new website for displaying Gold, Silver and Repo rates made using the best available web development tools.

Our Services

Gold Rates

Gold Rates on a particular day can be accessed and displayed very easily using this website. An auto update feature has also been integrated in the website which automatically updates the rates every 5 mins using the best and most reliable sources using APIs

Silver Rates

The Silver Rates for any given day can be quickly obtained and viewed on this page in a very straightforward manner. A function that automatically changes the rates once every five minutes based on the most reputable and trustworthy sources via APIs has also been incorporated into the website. This feature is known as an auto update.

Repo Rates

A page dedicated to displaying Repo Rates by RBI for the past 25 years has also been incorporated in the website. The fluctuations in the Repo Rates across years has been displayed usinng a graph in a better manner.

Graphical Analysis

The fluctuations in Gold and Silver rates for set period of time can be analysed graphically in the website. This feature has been implemented using a software known as Chart.JS . This feature can be accessed on the respective pages of Gold and Silver Rate Displays.
